## Overview
This project analyzes a retail **Superstore** dataset using **Google Sheets** (Excel equivalent).  
The data was cleaned, formatted, and explored through pivot tables and charts to uncover insights across:

- Product **Category**
- Customer **Segment**
- Geographical **Region**
- **Shipping Modes**
- Sales, Profit, and Profit Margin performance

---

## Objectives
- Clean and prepare the Superstore dataset  
- Standardize column formats (text, currency, percentage)  
- Create pivot tables for Category, Region/Segment, and Ship Mode  
- Build visualizations for Sales, Profit, and Profit Margin  
- Document all cleaning and analysis steps

---

## Tools Used
- **Google Sheets**
- Pivot Tables
- Formulas (Profit Margin, COUNTA)
- Chart Builder

---

## Data Cleaning Steps
- Uploaded Superstore dataset  
- Froze column headers for easier navigation  
- Removed duplicates (none found)  
- Checked key fields for blanks (2 blanks removed)  
- Created **Profit Margin** column  
- Applied correct data types (Text, Currency, Percentage)  
- Applied filters for sorting and reviewing data  
- Logged all cleaning steps in a separate **Data Cleaning Log** sheet

- **Consumer Segment** generated the highest total sales
